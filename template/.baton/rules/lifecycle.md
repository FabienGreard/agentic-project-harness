# Lifecycle

Management, Operations, and active Consultants are permanent top-level tasks. Contractors and Internal Audit are disposable. A new message to the exact permanent task is the sole wake event.

On wake, refresh once, complete safe actionable work, delegate through the authorized path, and record the next owner, action, dependency, and return trigger. Pause without polling when no meaningful action remains. Delegated idle is neither blocked nor complete.

Persistent goals are not role identity or lifecycle. Permanent roles never create, resume, recreate, inspect for control, or attach one. An automatic continuation without a new task message performs no work and is reported as stale host behavior. Host UI controls are not a lifecycle workaround.
