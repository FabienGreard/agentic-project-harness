# Canonical scenarios

Candidates receive only `inputs/<ID>.md`; independent judges receive the matching file under `oracles/`.

- H-001 — non-urgent change during active work
- H-002 — urgent overlapping change
- H-003 — independent parallel scopes
- H-004 — tightly coupled scope
- H-005 — specialist readiness missing
- H-006 — worker discovers missing intent
- H-007 — incomplete worker result
- H-008 — Delivery reaches idle
- H-009 — Director delegates and waits
- H-010 — human approval gate

Version input and oracle together. Never expose oracle directories to candidate contexts.
