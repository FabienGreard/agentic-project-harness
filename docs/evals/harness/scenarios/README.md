# Canonical scenarios

Candidates receive only `inputs/<ID>.md`; independent judges receive the matching file under `oracles/`.

- H-001 — non-urgent change during active work
- H-002 — urgent overlapping change
- H-003 — independent parallel scopes
- H-004 — tightly coupled scope
- H-005 — Consultant readiness missing
- H-006 — Contractor discovers missing intent
- H-007 — incomplete Contractor result
- H-008 — Operations reaches idle
- H-009 — Management delegates and waits
- H-010 — human approval gate
- H-011 — legacy persistent-goal auto-resume with full controls
- H-012 — risk-based initial review with mixed finding quality
- H-013 — bounded follow-up review and stopping condition

Version input and oracle together. Never expose oracle directories to candidate contexts.
