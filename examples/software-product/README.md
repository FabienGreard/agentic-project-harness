# Software Product preset

Management acts as **Product Manager**: owns user outcomes, product priorities, scope, readiness, and release decisions.

Operations acts as **Engineering Manager**: owns executable delivery, technical coordination, Contractor dispatch, integration, and verification.

The recommended starting Consultant is **Product Designer**, responsible for defining and accepting coherent user journeys, interaction behavior, and interface quality. The installer also offers Principal Engineer, Security Lead, Platform/SRE Lead, Data Lead, Accessibility Lead, and QA Lead. Select any number or none.

Operations may dispatch disposable Contractors for product engineering, platform and data, or verification and analysis. Those capability labels guide routing; they are not permanent user-managed roles.

Users still speak to Management, Operations, Consultants, and Contractors. The professional titles supply domain context without changing authority.
